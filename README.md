# PickCareRAG Repository


RESTful API를 통해 http 프로토콜을 경유하여 JAVA spring과 통신하는 RAG 서버 개발

1.문서 업로드 기능
2.문서 토크나이징 및 청킹
3.임베딩 벡터 변환
4.유사도 계산
5.검색기 구현
6.리랭커링 및 우선 순위 설정 프로세스 구현
7.MCP 서버 구현
8.최종 마무리
